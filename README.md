## 🎯 项目简介

Boss直聘自动招聘技能v7.1是一款**AI驱动的自动化招聘工具**，专为BOSS直聘企业端HR设计。

**一句话核心价值**：
> 让招聘变得更简单 - 自动筛选优质候选人，每次使用后更智能

### ✨ 核心亮点

- ⚡ **自动化招聘流程**：自动读取职位、解析要求、筛选候选人并打招呼
- 🧠 **智能自动复盘**：执行完成后自动分析优化，无需手动询问
- 📈 **持续进化机制**：每次使用后自我改进，越用越智能
- 🚀 **效率提升50-70%**：批量处理+智能评分，精准触达Top候选人

## ⚡ 核心功能

### 1. 智能缓存机制
- ✅ 避免重复解析职位（节省80%时间）
- ✅ 保留筛选配置（用户偏好持久保存）
- ✅ 增量更新（只处理新增/变更）

### 2. 批量筛选流程（10-15人/批次）
1. **硬性筛选**：学历、公司背景、专业
2. **智能评分**：综合多维度评估（100分制）
3. **优先联系**：Top 3-5候选人

### 3. 智能评分系统

| 维度 | 分数 | 评分规则 |
|------|------|----------|
| 学校层级 | 30分 | QS Top 50 > C9 > 985 > 211 |
| 公司匹配 | 40分 | 目标公司 > 相关大厂 > 其他 |
| 活跃度 | 15分 | 在线 > 1小时 > 今日活跃 |
| 求职意愿 | 15分 | 随时到岗 > 在职考虑 |

### 4. 完整遍历策略
- ✅ 滚动到"没有更多"提示才停止
- ✅ 候选人覆盖率从60-70% → **100%**
- ✅ 基于按钮状态100%准确识别已联系

## 🚀 安装使用

### 环境要求
- ✅ Claude Code SDK
- ✅ BOSS直聘企业账号

### 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/kinoxuan0518/bosszhibin-auto-recruiter-skill-v7.1.git

# 2. 进入目录
cd bosszhibin-auto-recruiter-skill-v7.1

# 3. 阅读主文档
cat SKILL-v7.1-evolution.md

# 4. 配置缓存目录
mkdir -p /home/claude/bosszhibin_cache
```

### 核心使用流程

```
1. 启动Skill
   ↓
2. 读取/解析职位（利用缓存）
   ↓
3. 确认筛选要求
   ↓
4. 执行筛选和联系 ← 【自动收集数据】
   ↓
5. 生成招聘总结报告
   ↓
6. 【新增】自动生成复盘报告 ← 无需询问！
   ↓
7. 【新增】更新进化历史
```

## 💡 使用建议

### 首次使用
1. 选择1-2个职位进行测试
2. 设置宽松的筛选条件（阈值60-70分）
3. 观察整个流程，了解系统行为
4. 根据结果调整筛选标准

### 日常使用
- 📅 **每天运行一次**（推荐上午10点或下午3点）
- 🎯 **处理所有高优先级职位**
- ⚡ **及时回复候选人消息**（24小时内）

### 高级技巧
- **动态调整阈值**：人才充足→80分，人才紧缺→60分
- **分时段处理**：上午高优先级，下午中优先级+新牛人

## ⚠️ 重要提醒

### 本技能的限制
- ❌ 无法绕过BOSS直聘的每日沟通上限
- ❌ 无法自动回复候选人的消息
- ❌ 需要人工审核最终简历和安排面试

### 本技能的价值
- ✅ 大幅提高简历筛选效率（**10倍以上**）
- ✅ 标准化筛选流程，减少人为偏差
- ✅ 及时触达新候选人，提高响应速度
- ✅ 数据化招聘过程，便于分析优化
